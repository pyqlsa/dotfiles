# Packages Overview

The **packages/** folder contains reusable libraries for the pi coding‑agent.

- **pi-config-utils** – helpers for reading `settings.json`.  *[see `pi-config-utils/README.md`](pi-config-utils/README.md)
- **pi-tools** – small CLI‑style tools (`searxng_search`, `web_scrape`).  *[see `pi-tools/README.md`](pi-tools/README.md)

*Installation*: All packages are already installed via Nix/Home‑Manager – no additional steps required.
